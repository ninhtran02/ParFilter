#  Reproducing the real data results in Section 5

## Section 5.1
1. To obtain the microarray data for the DSSIC case study, download and run the R script **GSE34553_DSS.R** and **GSE35651_DSS.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/DSSIC).
   
2. To run the replicability analysis and produce the plots in Section 5.1, download and run the R script **DSS.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/DSSIC)

## Section 5.2
1. To obtain the RNA-seq data for the Aire deletion case study, download the dataset R script **GSE151012.Rdata**, **GSE224247.R**, **GSE188870.R**, **GSE222285.R**, and **AIRE_data_maker.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/AIRE%20deletion). 

2. To run the replicability analysis and produce the plots in Section 5.2, download and run the R script **AIRE.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/AIRE%20deletion).
