#  Reproducing the real data results in Section 5

## Section 5.1
1. To obtain the microarray dataset for the DSSIC case study, download and run the R script **GSE34553_DSS.R** and **GSE35651_DSS.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/DSSIC).
   
2. To run the replicability analysis and produce the plots in Section 5.1, download and run the R script **DSS.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data/DSSIC)

## Section 5.2
1. Download the dataset R script **AIREKO_data.Rdata** and  **AIREKO_data_analysis_R.R** from [here](https://github.com/ninhtran02/ParFilter/tree/main/Real%20Data).

2. Simply run through the lines of **AIREKO_data_analysis_R.R** to perform the analysis and create the resulting plot.
